디자인 시스템 사용 가이드

이 디자인 시스템은 전남고등학교 교내 웹서비스의 일관된 사용자 경험을 보장하고 개발 효율성을 높이기 위해 제작되었습니다.

핵심 원칙: 컴포넌트의 구조적 무결성 유지

이 시스템의 스타일은 개별 클래스가 아닌, 특정 HTML 구조와 결합된 클래스들의 '집합'에 적용됩니다. 따라서 예제 코드의 구조를 임의로 변경하면 디자인이 깨질 수 있습니다.

  • 1. 구조가 곧 스타일입니다: .info-card 클래스는 내부에 <h2>와 <ul>이 있을 때를 가정하고 디자인되었습니다. 예제 코드의 HTML 구조와 클래스명을 하나의 세트로 사용해야만 의도된 디자인이 적용됩니다.
  • 2. 컴포넌트 단위로 사용하세요: 예제 코드를 '부품'처럼 분해하여 일부 클래스만 가져다 쓰지 마세요. 각 컴포넌트 블록을 하나의 '완성된 모듈'로 보고, 구조 전체를 가져가 사용하는 것을 원칙으로 합니다.
  • 3. 내용은 실제 데이터로 채우세요: 구조와 클래스는 유지하되, 예제 속 '주요 공지'와 같은 텍스트나 목록 아이템 등은 실제 데이터로 자유롭게 변경하여 사용하세요.

잘못된 사용 예시: .info-card 안에 완전히 다른 구조의 <div>나 <p> 태그를 넣는 경우
올바른 사용 예시: .info-card 예제의 HTML 구조를 그대로 가져와 내부 텍스트만 실제 공지사항 내용으로 바꾸는 경우

Colors

앱 전체의 일관된 톤앤매너를 유지하는 핵심 색상 팔레트입니다.

Primary--primary-color#0C7531
Background--bg-color#121212
Card Background--card-bg-color#1E1E1E
Text Primary--text-primary#E0E0E0
Text Secondary--text-secondary#A0A0A0
Border--border-color#333333

Typography

Noto Sans KR를 기반으로 한 타이포그래피 시스템입니다. 위계질서에 맞는 사용을 권장합니다.

h1 / Welcom Text - 22px / 700

h2 / Card Title - 18px

h3 / Meal Menu Title - 16px

h4 / D-Day Title - 14px

p / Body, Paragraph text - 14px ~ 15px

Link / More Link → span / Nav item text - 12px

Layout

모바일 환경에 최적화된 `480px` 너비의 중앙 정렬 레이아웃을 사용합니다.

.app-container {
    width: 100%;
    max-width: 480px;
    background-color: var(--bg-color);
    padding: 0 20px;
}

Icons

Font Awesome 6를 사용합니다. 아이콘은 명확한 의미 전달을 위해 사용하며, Primary 색상을 사용하는 것을 권장합니다.
